== Description ==
**[Contact Form 7 Range Slider](https://wordpress.org/plugins/range-slider-contact-form-7/)** is a easy to use setup field range in your contact form 7 for your [WordPress](https://en.wikipedia.org/wiki/WordPress) site.

**Contact Form 7 Range Slider** to make attractive design to multiple option on field

using **contact form 7 for slider** to setup digit number with min and max value on range slider. **Range slider in contact form 7** allow visitor to easy to slide number field to make attractive form in contact form 7. you can be put custom color as will prefix on left, right so it make more user friendly.
